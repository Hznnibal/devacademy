HTML - HyperText Markup Language



Objectifs

  • Comprendre le principe des documents HyperTextes
  • Connaître les différentes balises HTML5
  • Réaliser des documents HTML simples



Pré-requis

Pour réaliser les différents exercices, vous aurez besoin d'utiliser un éditeur de texte.

Utilisez de préférence Visual Studio Code. Si ce n'est pas déjà fait, rendez-vous sur le site officiel : https://code.visualstudio.com/ pour le télécharger (choisir le téléchargement correspondant à votre système d'exploitation).



Cheminement

Demandez à votre formateur de vous faire une présentation de cette séance.

Vous y découvrirez les différents outils à utiliser ainsi que quelques explications sur le langage de balisage HTML.

Ensuite, lisez les supports puis réalisez les exercices au fur et à mesure.


Phase 0 - Structurer mes dossiers et fichiers

Avant toute chose, veuillez créer l'aborescence ci-dessous, en renommant le dossier stagiaire 1 par votre prénom,

  • Si vous êtes sur Linux : positionnez le répertoire là où vous le souhaitez sur la machine (nous le synchroniserons en ligne plus tard avec Git)
  • Si vous êtes sur Windows : créez votre répertoire dans le dossier partagé sur le réseau, nommé AFPA GRP-XX (voir éventuellement avec votre formateur pour le numéro du groupe)

Vous y rangerez les exercices et éventuelles évaluations attendues.
